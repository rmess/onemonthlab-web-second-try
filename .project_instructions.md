# Déclic Agency - Project Instructions

## Overview
Landing page haute conversion pour l'agence web Déclic basée à Sète, France. Site en français, ciblant les petits artisans/commerçants.

## Brand Identity
- **Colors**: Coral-orange accent (#E8734A / warm-orange), Deep navy text (#1E293B), Soft cream backgrounds
- **Tone**: Chaleureux, direct, humain, rassurant, anti-jargon
- **Fonts**: DM Sans (body), Plus Jakarta Sans (headings)
- **Language**: French only (lang="fr")

## CSS Variables (custom)
- primary: warm coral-orange (hsl 16 78% 60%)
- background: soft warm white
- foreground: deep navy

## Architecture
- Single-page landing with 9 sections + admin dashboard (/admin)
- Auth: admin-only for managing contacts & bookings
- DB: Contact + Booking models
- Notifications: contact form + booking confirmation emails

## Key Patterns
- All content in French
- No dark mode (warm brand, light only)
- Framer Motion for scroll animations
- Zod validation on forms
- Calendar booking system for diagnostic gratuit
